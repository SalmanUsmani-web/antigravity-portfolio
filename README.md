# Anti-Gravity 3D Portfolio

A modern and responsive personal portfolio website with an interactive 3D anti-gravity design.

## Live Demo

[View Live Portfolio](https://antigravity-portfolio-chi.vercel.app/)

## Features

* Interactive 3D background using Three.js
* Smooth animations and typing effect
* Responsive design for mobile, tablet, and desktop
* About, Skills, Education, Training, Projects, and Contact sections
* Resume download option
* Contact information
* Modern dark user interface

## Technologies Used

* HTML5
* CSS3
* JavaScript
* Three.js
* Vite
* Git and GitHub
* Vercel

## Run Locally

Clone the repository:

```bash
git clone https://github.com/SalmanUsmani-web/antigravity-portfolio.git
```

Open the project folder:

```bash
cd antigravity-portfolio
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Then open:

```text
http://localhost:5173/
```
